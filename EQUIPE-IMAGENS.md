# Fotos da equipe – página Equipe

Salve cada foto em **`public/images/equipe/`** com o **nome exato** abaixo. Depois rode:

```bash
npm install
npm run optimize-equipe
```

O script redimensiona (máx. 400×400), comprime (JPEG qualidade 82) e sobrescreve os arquivos.
